# 🎮 Как работает мини-игра по взлому замков

## Простыми словами (для начинающих)

### Что я сделал:
1. **Добавил мини-игру** в стиле Fallout — когда игрок использует отмычку на заблокированном сундуке, запускается игра
2. **Без GUI** — всё происходит через чат (сообщения в игре)
3. **Механика:** сервер загадывает последовательность направлений (вверх/вниз/влево/вправо), игрок должен её угадать

---

## Как это работает (пошагово):

### Шаг 1: Начало
```
Игрок берёт отмычку → ПКМ по сундуку с замком
         ↓
Сервер создаёт последовательность (например: ↑↓←→)
         ↓
Игрок видит: "Тип замка: IRON, Длина: 4, Попыток: 4"
```

### Шаг 2: Ввод
```
Игрок пишет в чат: /lockpick up
         ↓
Правильно? → Зелёный звук + [↑][?][?][?]
Неправильно? → Красный звук + "Последовательность сброшена!"
```

### Шаг 3: Результат
```
ВСЕ введены правильно → Сундук ОТКРЫВАЕТСЯ + замок снимается
Кончились попытки → Отмычка ломается, замок остаётся
```

## Как тестировать:

### 1. Запустите мод:
```bash
gradlew runClient
```

### 2. В игре:
1. Создайте сундук
2. Создайте замок (например, `/lock give @p wooden_lock`)
3. ПКМ по сундуку с замком → замок установлен
4. Создайте отмычку (`/lock give @p lockpick`)
5. ПКМ по сундуку с отмычкой → мини-игра началась!
6. Вводите: `/lockpick up`, `/lockpick down`, etc.

---

## Что нужно знать:

### 📌 Команды:
- `/lockpick up` — ввести "вверх"
- `/lockpick down` — ввести "вниз"
- `/lockpick left` — ввести "влево"
- `/lockpick right` — ввести "вправо"
- `/lockpick cancel` — отменить

### 📌 Сокращения:
- `/lockpick u` или `/lockpick ↑` или `/lockpick w` — тоже "вверх"
- Аналогично для других направлений

### 📌 Сложность:
| Замок | Длина | Попытки |
|-------|-------|---------|
| Деревянный | 3 | 5 |
| Железный | 4 | 4 |
| Золотой | 5 | 3 |
| Алмазный | 6 | 2 |

## Если что-то не работает:

### ❌ Ошибка: "Вы не взламываете замок!"
**Причина:** Вы не начали взлом  
**Решение:** Используйте отмычку на сундуке

### ❌ Ошибка: "Вы уже взламываете замок!"
**Причина:** Вы уже взламываете другой сундук  
**Решение:** `/lockpick cancel`

### ❌ Отмычка не работает
**Причина:** Сундук не заблокирован  
**Решение:** Сначала установите замок
