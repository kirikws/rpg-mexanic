# 🎮 Мини-игра по взлому замков (Fallout-style)

## Описание
Мини-игра по взлому замков в стиле Fallout **без GUI** — используется только чат и сообщения игроку.

---

## 📖 Как это работает

### 1. Начало взлома
1. Возьмите **отмычку** (`Lockpick`) в руку
2. Нажмите **ПКМ** по заблокированному сундуку
3. Одна отмычка удалится из инвентаря
4. В чате появится сообщение с инструкцией

### 2. Механика мини-игры

#### Последовательность направлений
Сервер генерирует случайную последовательность направлений:
- **up** (вверх) — ↑
- **down** (вниз) — ↓
- **left** (влево) — ←
- **right** (вправо) — →

#### Ввод команд
Игрок должен ввести команды в чат:

```
/lockpick up
/lockpick down
/lockpick left
/lockpick right
```

Или использовать сокращения:
```
/lockpick u      (или /lockpick ↑, /lockpick w)
/lockpick d      (или /lockpick ↓, /lockpick s)
/lockpick l      (или /lockpick ←, /lockpick a)
/lockpick r      (или /lockpick →, /lockpick d)
```

#### Прогресс-бар
После каждого правильного ввода показывается прогресс:
```
[↑][↓][?][?][?]  — нужно ввести ещё 3 направления
```

#### "Горячо/холодно" (визуальная подсказка)
- **Зелёный цвет** — правильный ввод ✅
- **Красный цвет** — неправильный ввод ❌
- **Звук высокой частоты** — правильно
- **Звук низкой частоты** — неправильно

### 3. Результат

#### ✅ Успех
- Все направления введены правильно
- Сундук **автоматически открывается**
- Замок **снимается** (сундук теперь без замка!)
- Звук открытия двери

#### ❌ Провал
- Игрок ошибся 3+ раза (зависит от типа замка)
- Отмычка **ломается**
- Замок **остаётся** на сундуке
- Нужно пробовать снова с новой отмычкой

|-----------|-------------------------|---------|

| **Железный** (Iron) | 4 | 4 |
| **Золотой** (Golden) | 5 | 3 |

## 🎯 Пример игры

### Сценарий: Взлом железного замка

**1. Начало:**
```
Игрок: (ПКМ по сундуку с отмычкой)

Сервер: ═══════════════════════════════
Сервер: 🔓 ВЗЛОМ ЗАМКА
Сервер: Тип замка: IRON
Сервер: Длина последовательности: 4
Сервер: Попыток осталось: 4
Сервер: 
Сервер: Используйте:
Сервер:   /lockpick up    или /lockpick ↑
Сервер:   /lockpick down  или /lockpick ↓
Сервер:   /lockpick left  или /lockpick ←
Сервер:   /lockpick right или /lockpick →
Сервер: ═══════════════════════════════
```

**2. Ввод команд:**
```
Игрок: /lockpick up
Сервер: [↑][?][?][?]
Сервер: Осталось: 3

Игрок: /lockpick down
Сервер: [↑][↓][?][?]
Сервер: Осталось: 2

Игрок: /lockpick left
Сервер: [↑][↓][←][?]
Сервер: Осталось: 1

Игрок: /lockpick right
Сервер: ✓ Замок взломан!
(Сундук открывается автоматически)
```

**3. Если ошибка:**
```
Игрок: /lockpick up
Сервер: [↑][?][?][?]
Сервер: Осталось: 3

Игрок: /lockpick right  (неправильно!)
Сервер: ✗ Неверно! Попробуйте снова.
Сервер: Попыток осталось: 3
Сервер: Последовательность сброшена! Начните сначала.
```

## 🎮 Стратегии

### Для деревянных замков (3 направления, 5 попыток)
- Легко, можно угадать даже без стратегии
- Вероятность успеха: ~75%

### Для железных замков (4 направления, 4 попытки)
- Нужна базовая стратегия
- Вероятность успеха: ~50%

### Для золотых замков (5 направлений, 3 попытки)
- Сложно, требуется удача
- Вероятность успеха: ~25%

### Для алмазных замков (6 направлений, 2 попытки)
- Очень сложно, почти невозможно без удачи
- Вероятность успеха: ~10%

## 🔧 Команды

### `/lockpick <direction>`
Ввести направление в мини-игре.

**Аргументы:**
- `up`, `down`, `left`, `right` — полные названия
- `u`, `d`, `l`, `r` — сокращения
- `↑`, `↓`, `←`, `→` — символы
- `w`, `s`, `a`, `d` — WASD

### `/lockpick cancel`
Отменить мини-игру (можно начать заново с новой отмычкой).

---

## 💡 Советы

1. **Запоминайте звуки** — высокий звук = правильно, низкий = неправильно
2. **Следите за цветом сообщений** — зелёный = хорошо, красный = плохо
3. **Будьте осторожны с алмазными замками** — всего 2 попытки!
4. **Используйте ключ**, если он у вас есть — это проще, чем взламывать

## 🎨 Визуальные эффекты

| Событие | Эффект |
|---------|--------|
| Начало взлома | Звук рычага + золотое сообщение |
| Правильный ввод | Зелёный прогресс-бар + высокий звук |
| Неправильный ввод | Красное сообщение + низкий звук |
| Успех | Зелёное жирное сообщение + звук двери |
| Провал | Красное жирное сообщение + звук поломки |

## 🐛 Возможные проблемы

### "Вы не взламываете замок!"
- Вы используете команду `/lockpick`, но не начинали взлом
- Решение: используйте отмычку на сундуке сначала

### "Вы уже взламываете замок!"
- Вы уже взламываете другой сундук
- Решение: `/lockpick cancel` или дождитесь текущего

### Отмычка не работает
- Сундук не заблокирован
- Решение: сначала установите замок

| `ENTITY_ITEM_BREAK` | Провал |

## 📝 Технические детали

### Структура мини-игры
- **LockpickMinigame** — главный класс
- **LockpickSession** — сессия для каждого игрока
- Хранится в памяти (HashMap)
- Сессия удаляется после завершения

### Интеграция
- `ChestBlockMixin` — запускает мини-игру при ПКМ отмычкой
- `LockCommands` — обрабатывает команды `/lockpick`
- `LockManager` — удаляет замок при успехе
